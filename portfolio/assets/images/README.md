# Portfolio Assets

## Images Folder

Thư mục này chứa các hình ảnh cho portfolio của bạn.

### Các file ảnh cần thêm:

1. **profile.jpg** - Ảnh đại diện của bạn (400x400px, hình vuông)
2. **about.jpg** - Ảnh cho phần giới thiệu
3. **project1.jpg** - Ảnh dự án 1 (800x600px)
4. **project2.jpg** - Ảnh dự án 2 (800x600px)
5. **project3.jpg** - Ảnh dự án 3 (800x600px)
6. **project4.jpg** - Ảnh dự án 4 (800x600px)
7. **project5.jpg** - Ảnh dự án 5 (800x600px)
8. **project6.jpg** - Ảnh dự án 6 (800x600px)

### Hướng dẫn:

- Thêm ảnh của bạn vào thư mục này
- Đảm bảo tên file khớp với tên trong code
- Kích thước đề xuất để hiển thị tốt nhất
- Có thể sử dụng ảnh placeholder từ https://placeholder.com hoặc https://unsplash.com

### Ảnh placeholder tạm thời:

Bạn có thể sử dụng các URL sau trong HTML để test:
- Profile: https://via.placeholder.com/400
- Projects: https://via.placeholder.com/800x600
